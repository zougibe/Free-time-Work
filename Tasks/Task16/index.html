Workflow Canvas

ML Workflow Visualizer Guide

Getting Started

Drag components from the sidebar to create your machine learning workflow. Connect them to define the data flow between steps.

Adding Nodes

Drag and drop components from the sidebar onto the canvas to create your ML workflow steps.

Connecting Nodes

  • Right-click to select a node (yellow outline)
  • Right-click another node to connect them

Configuring Nodes

Left-click on any node to edit its name, parameters, and add documentation.

Moving Nodes

Drag nodes to reposition them. Connections will automatically update to follow the nodes.

Pro Tip

Hover over nodes to see quick details. Double-click to quickly edit node names.

Confirm Clear

Are you sure you want to clear all nodes and connections? This action cannot be undone.
