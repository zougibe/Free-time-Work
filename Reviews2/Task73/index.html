How to Use

Select a supplier to view their performance metrics (Timeliness, Quality, Cost) in the summary card, radar chart, and comparison table. Choose "None" to clear the main selection. Double-click a table row to toggle highlighting a supplier, compare suppliers via checkboxes, sort the table, or export as CSV. Clear comparisons to reset highlights, or copy the summary for reports. Toggle dark mode for better viewing.
